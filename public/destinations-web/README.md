# 365tours.in Destination Catalog

A complete, per-destination catalog of every place on **365tours.in**, built for the
site revamp. Generated by [`scripts/build_catalog.py`](../../scripts/build_catalog.py)
(re-runnable; already-downloaded files are skipped).

## What's here

- **91 destinations** — one folder each: `<name-slug>-<id>/` (e.g. `vietnam-365T_00002/`)
- **879 image assets** — gallery photos, recipe-book image, itinerary cards, and country flag
- **`catalog.json`** — the master index (array of 91 destination records)
- **`manifest.json`** — a flat list of every image (one row per file)
- Each destination folder also has its own **`metadata.json`**

Every place keeps its stable source ID (`365T_000XX`), which is how the live site
identifies destinations (`destination-detail.php?id=365T_000XX`) and organizes assets
(`tour_images/365T_000XX/`).

## `catalog.json` record shape

```json
{
  "id": "365T_00002",
  "name": "Vietnam",
  "display_name": "VIETNAM",
  "slug": "vietnam-365T_00002",
  "detail_url": "https://365tours.in/destination-detail.php?id=365T_00002",
  "flag_file": "VIETNAM_flag.png",
  "signature_itineraries": null,
  "destinations_offered": ["Hanoi", "Hoi An", "Mekong Delta", "..."],
  "folder": "public/destinations/vietnam-365T_00002",
  "image_count": 10,
  "images": [
    { "file": "365T_00002_24.jpg",
      "path": "public/destinations/vietnam-365T_00002/365T_00002_24.jpg",
      "source_url": "https://365tours.in/tour_images/365T_00002/365T_00002_24.jpg" }
  ]
}
```

## Image naming on the source

- `365T_000XX_NN.jpg` — destination gallery / signature-itinerary photos
- `itenary_NN.jpg` — itinerary card images (used by some destinations)
- `recipebook_img.jpg` — the "recipe book" feature image
- `<COUNTRY>_flag.png` — country flag

## Known source-data gaps (not scrape errors)

- **7 destinations have no "Destinations Offered" list** because the live page's list
  is empty: Antarctica, Cyprus, Macau, Maldives, Mauritius, Singapore, West Indies.
- **2 itinerary images 404 on the server** (referenced but missing):
  `365T_00036/itenary_1.jpg`, `365T_00066/itenary_68.jpg`.
- `signature_itineraries` is often `null` — the count is not consistently present in the
  page markup.
